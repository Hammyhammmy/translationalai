LightScribe

AI-powered medical scribe deployed in production.
Designed by a Canadian physician.

How It Works

LightScribe uses LightProof SDK to generate medical documentation with verifiable source attribution. The tool processes clinical input and produces structured documentation while mapping each statement to source material.

The interface is designed for clinical workflows with three primary actions: Start, Finish, and Copy. A text box allows for private inputs during patient interactions, enabling real-time documentation support.

Features

  • Generates consult notes, SOAP notes, and patient summaries
  • Custom templates for various clinical scenarios
  • Verifiable outputs with source attribution
  • Simple interface optimized for clinical use
  • Private input field for real-time documentation during patient encounters

Security and Privacy

  • End-to-end encryption for data in transit and at rest
  • Canadian data sovereignty—all data stored within Canada
  • Aligned with PIPEDA privacy principles and provincial privacy requirements
  • Self-assessed alignment with HIPAA security protocols (not certified)
  • Privacy by Design approach—automatically filters PHI
  • No data used for AI training

Status

LightScribe is currently deployed in real clinical environments, demonstrating the practical value of verifiable AI in healthcare documentation workflows. Available for pilot use with select institutions.
